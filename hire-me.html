DEVOPS & CLOUD JOURNEY

From terminal to cloud — my path into DevOps.

I'm Yogender Singh. Here's the honest story of how I got hands-on with Linux, Git & GitHub, Docker, Kubernetes, and AWS — and why I think that makes me worth a conversation.

01 // THE STACK

How I built this foundation

Five stops, in the order I actually learned them.

01 / 05

Linux

Getting comfortable at the command line

Every DevOps journey starts here. I learned to navigate the file system, manage users and permissions, work with processes, and automate small tasks with shell scripts — the muscle memory that everything else is built on.

02 / 05

Git & GitHub

Learning to collaborate like a team

Version control turned solo scripts into shareable, trackable work. I practiced branching, merging, resolving conflicts, and using pull requests the way real engineering teams do.

03 / 05

Docker

Packaging software so it runs anywhere

Containers solved the "works on my machine" problem. I learned to write Dockerfiles, build images, and run containers so applications behave the same in development and in production.

04 / 05

Kubernetes

Scaling from one container to many

Once one container was easy, I went looking for the next problem: running many of them reliably. Kubernetes taught me deployments, services, and how self-healing infrastructure actually behaves under change.

05 / 05

AWS

Taking it all to the cloud

The final piece was learning to provision and deploy on real cloud infrastructure with AWS — connecting everything I'd learned locally to systems that can actually serve users.

02 // WHY WORK WITH ME

What I bring to a team

Hands-on by default

I don't stop at tutorials — I rebuild what I learn until it's muscle memory, not just theory.

Automation-first thinking

If I'm doing something twice, I'm already looking for a way to script or automate it.

Comfortable with the full stack of tools

From the Linux shell to container orchestration to the cloud — I can move across the layers a DevOps role actually touches.

A genuine learner's mindset

I'm early in this journey and I say so — but I close gaps fast, and I'm looking for a team to grow with.

Ready to talk?

I'm open to DevOps, Cloud, and Infrastructure roles — reach out any way that's easiest for you.
